昇腾 AI 异构
计算架构

CANN（Compute Architecture for Neural Networks）是华为针对 AI 场景推出的异构计算架构，全面支持昇腾 NPU，为 AI 应用开发者提供从算子开发到网络部署的完整工具链。

import ascendcl
import numpy as np

# 初始化 CANN 运行环境
ret = ascendcl.init()
context = ascendcl.rt.create_context(0)

# 分配设备内存
device_ptr, ret = ascendcl.rt.malloc(1024, 0)

# 数据传输
ascendcl.rt.memcpy(device_ptr, host_ptr, 1024, 1)

核心特性

为 AI 开发者提供完整的异构计算工具链

A

AscendCL 编程接口

类 CUDA 编程接口，支持 C/C++ 开发，提供设备管理、内存管理、算子调用等完整 API。

T

TBE 算子开发

基于 Python 的算子开发框架，支持自定义算子，提供丰富的 DSL 和调试工具。

M

MindSpore 深度集成

与 MindSpore 框架深度集成，支持训练和推理全流程，提供最佳性能。

C

模型转换优化

ATC 工具支持 TensorFlow、PyTorch、ONNX 等主流框架模型转换。

P

性能分析工具

完整的性能分析和调优工具链，帮助开发者快速定位性能瓶颈。

D

开发者生态

丰富的文档、示例代码和社区支持，助力开发者快速上手。
